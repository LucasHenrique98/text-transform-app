<h1 align='center'>Text Transformer App</h1>

<p align='center'>Este app foi desenvolvimento com base nas aulas do Bootcamp Full Stack do IGTI. Seu objetivo é utilizar o valor de um campo Input e replica-lo em outros campos e transforma-lo conforme os requisitos do exercício. </p>



<p>Foram utilizados métodos de array e string para a manipulação dos valores inseridos e Class Components para os Components</p>



:hammer_and_wrench:  Tecnologias utilizadas:

- React
- Class Componentes
- CSS3
